Computational Color Theory

Beautiful UI themes generated by algorithms, not hand-picked colors

Golden Ratio Color Generation

Uses the golden ratio (φ = 1.618) to create harmonious hue relationships. Each color is offset by 137.5° (360°/φ) for mathematically perfect harmony.

Triadic Harmony System

Creates themes using colors evenly spaced around the color wheel (120° apart). Generates balanced, vibrant themes with natural contrast.

Analogous Gradient Flow

Creates smooth color transitions using adjacent hues. Perfect for creating cohesive themes with subtle variations and natural flow.

Split Complementary Dynamics

Uses a base color with the two colors adjacent to its complement. Creates high contrast while maintaining harmony.

Perceptual Luminance Scaling

Generates themes based on human perception of brightness. Uses CIE LAB color space for perceptually uniform color transitions.

Fibonacci Spiral Progression

Maps colors along a Fibonacci spiral in color space. Creates organic, naturally pleasing color relationships found in nature.
